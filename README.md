# 🎮 Empresário em Ação

**Empresário em Ação** é um jogo educativo desenvolvido para o **Conecta Expo**, evento realizado em Goianinha/RN, com o objetivo de ensinar conceitos de empreendedorismo, contabilidade, gestão financeira e reforma tributária de forma interativa e divertida.

O jogador assume o papel de um empreendedor que está iniciando seu negócio e, ao longo da jornada, enfrenta desafios, conversa com especialistas e responde perguntas para avançar pelas fases.

---

## 🚀 Funcionalidades

- 🎮 Jogo desenvolvido em JavaScript utilizando Phaser 3
- 👤 Seleção de personagem
- 📖 História com diálogos entre os personagens
- 🏢 3 fases com progressão de dificuldade
- 👨‍💼 NPCs e chefes (Bosses) personalizados
- ❓ Sistema de perguntas e respostas
- 🔀 Perguntas e alternativas embaralhadas automaticamente
- ⏱️ Cronômetro para cada pergunta
- 💬 Explicação após cada resposta
- ❤️ Sistema de vidas
- ⭐ Sistema de pontuação
- 🏆 Ranking Top 10
- 🎵 Efeitos sonoros e música de fundo
- 📱 Compatível com computador e dispositivos touch

---

## 📚 Conteúdo abordado

Durante o jogo são trabalhados temas como:

- Abertura de empresas
- CNPJ e MEI
- Nota fiscal
- Regime tributário
- Organização financeira
- Balanço Patrimonial
- DRE
- Pró-labore
- Simples Nacional
- Reforma Tributária
- CBS e IBS
- SPED Contábil (ECD)
- Goodwill (Ágio)
- Consolidação das demonstrações financeiras

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Phaser 3

---

## 🎯 Objetivo

O projeto foi desenvolvido para demonstrar como a **gamificação** pode tornar o aprendizado de temas empresariais e contábeis mais acessível, proporcionando uma experiência dinâmica e educativa para os participantes do evento.
